This is a simple TypeScript Express Boilerplate Template

## Useful Commands

##### Install Dependencies

```bash
npm install
```

##### Run development server

```bash
npm run dev
```

##### Run Unit Testings

```bash
npm run test
```

##### Build and run the production server

```bash
npm run build
npm run start
```
